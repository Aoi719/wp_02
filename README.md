# ブログサイト

## 仕様

- フォント　　：「Noto Sans」と「Noto Sans JP」を使用
- コンテンツ　：最大幅は1200pxで横のパディングは16px
- ヘッダー　　：全幅で固定。グロナビメニューは1200pxの中におさめる
- ピックアップ：記事3つを横並びに配置。リンクはホバー時に下線が消える

## 制作時間

### 静的ページ構築

- TOPページ：3h  
  ※記事詳細等はWordPressテーマ開発時に直接構築したため、上記工数には含めていません。

- WordPressテーマ構築：8h  
  ※記事詳細等の構築も含む
